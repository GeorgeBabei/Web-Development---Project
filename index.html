02. About myself

Hi, I'm George! I'm a structural design engineer who likes puzzles, legos and video games. Some time ago, I decided to make a career change and turned my attention to the thing that always seemed impossible to me: programming. Fast forward, I discovered how fun, challenging and rewarding is to write code, and , because I like to see my work come to life, I've decided to learn Web Development.

So far, I have used the following technologies:

  • HTML + CSS

  • React

I'm also planning to learn some Back-end stuff, little by little, while continuing to develop my Front-end skills!
